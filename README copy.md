### 安装

   1. 克隆项目，进入项目目录安装依赖

      ```
      git clone https://gitee.com/suancaixianyu/vue3-schub.git
      cd vue3-schub
      pnpm i
      ```

   2. 启动项目

      ```
      npm run dev
      ```

### 扩展指令

   1. 格式化代码

      ```
      npm run prettier
      ```
   2. 打包项目

      ```
      npm run prettier
      ```

### 开发环境

我这使用 `vscode` ，其他开发环境自行研究

`vscode`建议安装以下插件进行开发：

1. Chinese (Simplified)
2. JavaScript and TypeScript Nightly
3. TypeScript Vue Plugin (Volar)
4. Vue Language Features (Volar)
5. VS Code for Vite
